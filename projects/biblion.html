Biblion

Ik heb dit project gedaan voor NBD-Biblion. De opdracht was om een prototype in elkaar te zetten van een boek-zoeker, waarmee bezoekers van een bibliotheek snel hun boeken konden zoeken. Bij dit project heb ik verschillende ontwerp begrippen geleerd en toegepast, en heb ik geleerd hoe ik een werkend prototype kan bouwen.
